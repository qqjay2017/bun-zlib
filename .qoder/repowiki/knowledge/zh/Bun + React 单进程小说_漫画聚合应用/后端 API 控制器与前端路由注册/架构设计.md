本模块由两部分组成：controllers/ 目录下的 Bun HTTP 控制器层（book、cache、download、source 四个文件按业务域拆分），统一通过 defineController(prefix, handlers) 以字符串键 METHOD /path 声明式注册路由；lib/router.ts 是前端路由装配入口，使用 TanStack React Router 的 createRouter + addChildren 将 routes/ 下的页面路由组合成完整树，并在根路径 / 重定向到 /novel，同时注入全局 queryClient 上下文。依赖方向单向：控制器调用 backend.ts、lib/cache-manager、lib/download-manager、lib/source-config 等后端能力，router.ts 仅消费 routes/ 组件，不反向依赖 controllers。
