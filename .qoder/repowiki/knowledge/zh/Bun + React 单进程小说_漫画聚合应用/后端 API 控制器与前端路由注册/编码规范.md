- 控制器统一通过 defineController(prefix, { "METHOD /path": handler }) 字符串键声明式注册，避免手写路由表。
- 所有 JSON 响应采用 { success: boolean, data?: any, error?: string } 统一信封结构，错误时附带 HTTP status。
- REST 资源路径采用层级化前缀分组：/api、/api/cache、/api/download，并按 type/sourceId/bookId 三段式参数组织缓存读取接口。
- 控制器只负责参数解析与响应封装，业务逻辑委托给 lib/*-manager 或 backend.ts，保持薄控制器风格。
