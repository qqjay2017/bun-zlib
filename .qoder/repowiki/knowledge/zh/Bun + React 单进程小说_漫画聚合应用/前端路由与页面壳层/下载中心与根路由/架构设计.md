本模块由 TanStack Router 的声明式路由文件组成：`routes/__root.tsx` 通过 `createRootRoute` 注册根路由，挂载包含「小说/漫画/下载」三个 Tab 的 `RootLayout` 组件，并在开发环境注入 `TanStackRouterDevtools`；`routes/download.tsx` 以 `getParentRoute: () => rootRoute` 挂入 `/download` 路径，导出 `DownloadCenter` 组件作为页面入口；`components/not-found.tsx` 暴露 `NotFound` 组件，被根路由的 `notFoundComponent` 消费。依赖方向单向：子路由 → 根路由 → 共享组件，无反向引用。下载功能采用 REST + SSE 混合模式：初始列表通过 `GET /api/download/tasks` 拉取，后续进度增量更新通过 `EventSource('/api/download/progress')` 推送，取消任务使用 `DELETE /api/download/:taskId`。
